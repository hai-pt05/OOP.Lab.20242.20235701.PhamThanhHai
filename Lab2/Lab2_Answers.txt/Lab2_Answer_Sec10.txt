Question:
    - If you create a constructor method to build a DVD by title then create a constructor method to build
a DVD by category. Does JAVA allow you to do this?

Answer:
    - Trong JAVA có cho phép định nghĩa nhiều constructor gọi là overloading, nhưng với điều kiện là
các constructor phải có danh sách tham số khác nhau về số lượng hoặc kiểu dữ liệu để trình biên dịch có thể phân biệt được chúng.

    - Còn khi tạo constructor để tạo DVD theo Title và 1 constructor nữa để tạo DVD theo Category thì lại không thể vì cả hai có cùng
kiểu dữ liệu String. Ví dụ trong code sau:

public class DigitalVideoDisc {
    private String title;
    private String category;

    // Constructor chỉ dùng title
    public DigitalVideoDisc(String title) {
        this.title = title;
    }

    // Constructor chỉ dùng category
    public DigitalVideoDisc(String category) {
        this.category = category;
    }
}

Java báo lỗi vì cả hai constructor đều nhận một tham số kiểu String.
VD: new DigitalVideoDisc("The Matrix"). Java không biết bạn muốn gán "The Matrix" cho title hay category.
    
    - Để đúng thì cần có thêm tham số khác của constructor của category để phân biệt

public class DigitalVideoDisc {
    private String title;
    private String category;

    // Constructor dùng title
    public DigitalVideoDisc(String title) {
        this.title = title;
    }

    // Constructor dùng category và thêm một tham số khác để phân biệt
    public DigitalVideoDisc(String category, int dummy) {
        this.category = category;
    }
}
